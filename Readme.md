Source code Demo APP Tiêm chủng.
1. Back-end: C# .Net 8
Pattern: Clean Architecture

2. Front-end: React-JS.
---
1. Build run: back-end:
cd vào Back-end -> chạy dotnet build
Xong cd vào Back-end/VNVC.WebAPI -> chạy dotnet run để chạy API trên Localhost

2. Build run: front-end
cd vào Front-end -> chạy npm install
Xong chạy npm run build
Xong nếu ko lỗi gì -> chạy npm run dev để chạy web-app React trên Localhost.

2. Script DB là File ScriptDB.SQL
- Tạo database trên SQL Server xong chạy Script để sinh Data: có đầy đỏ Data để Test
- Sử ConnectionString ở file appsettings.json trong VNVC.WebAPI
